# **App Name**: BrandBoost Landing

## Core Features:

- Header: Sticky header with logo, navigation anchors, and primary CTA.
- Hero Section: Hero section with engaging headline, concise sub-headline, CTAs, and trust indicators.
- Founder Intro: Founder section featuring a short introduction and key achievements.
- Value Proposition: Why Us section with cards highlighting the agency's unique value propositions.
- Service Listing: Services section showcasing service offerings with clear pricing in USD.
- Package Customizer: Package Builder for dynamic pricing and customized service bundles with localStorage persistence.
- Lead Generation: Lead capture modal triggered by primary CTAs, integrating with Supabase and Telegram.

## Style Guidelines:

- Primary color: Ocean Blue (#00C9FD) for main calls to action and links.
- Background color: Sky Blue (#ADFFFE), a light desaturated tint of the primary, applied to section backgrounds.
- Accent color: A desaturated yellow-green (#B1FF00), providing contrast on headers and details. (#B1FF00)
- Headings: 'Poppins' (sans-serif) in Bold/ExtraBold for headlines to maintain a modern, fashion-forward appearance.
- Body: 'General Sans' (sans-serif) in Regular/Medium for body text.
- Sleek and modern icons from the Cruip UI design system, following the rounded style, soft shadows, and generous spacing.
- Utilize Cruip UI design system's layout structure: sleek cards, rounded corners (12-16px), soft shadows, and generous spacing for a modern aesthetic.
- CSS-only fade/slide animations for subtle transitions. Avoid heavy libraries and framer-motion to maintain performance.
